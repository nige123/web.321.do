# 321 gobin Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `321 gobin make | release | rollback | status` command that builds, signs, and releases cross-arch Go binaries and produces the `version.json` manifest the self-updater consumes.

**Architecture:** A thin `Deploy::Command::gobin` dispatches to a testable `Deploy::GoBin` pipeline library. The two external boundaries - the GoReleaser runner and the S3 client - sit behind injectable seams (duck-typed objects), so `make`/`release`/`rollback` are drivable end-to-end in tests with a fake runner and fake S3. All version math, config generation, and manifest mutation is pure and unit-tested without network or toolchain. `Deploy::Manifest`/`Deploy::Config` are extended to surface the `gobin:` block.

**Tech Stack:** Perl 5.42 (`Mojo::Base -base, -signatures`), `Mojo::JSON` (canonical encode for stable manifests), `YAML::XS` (already a dep, for `.goreleaser.yaml`), the `aws` CLI (S3), GoReleaser + Go (build), ed25519 signing. Tests: `Test::More` + `Path::Tiny` tempdirs, following t/48 / t/10 idioms.

## Global Constraints

- Four-space indentation; `Mojo::Base -base, -signatures` for libs, `Mojo::Base 'Deploy::Command', -signatures` for the command.
- **No copyright header** on 321's own lib/bin files (match the existing repo; the header is only for generated service code in skill templates).
- **No em/en dashes** anywhere - plain ` - `. A house rule and there are mojibake tests elsewhere.
- JSON responses / data shapes follow `Mojo::JSON`; `encode_json` sorts keys, giving reproducible `version.json`.
- Commands auto-register via `app->commands->namespaces` = `Deploy::Command` - a new `Deploy::Command::gobin` is discovered with no wiring.
- Signing key + S3 creds come from `<repo>/conf/secrets.conf` (a Perl hashref file, gitignored per-host). The key is **never logged**.
- `CGO_ENABLED=0` for every build; version stamped via ldflags into `gobin.version_var`.
- Default target matrix (when `gobin.targets` is absent): `linux/amd64`, `linux/arm64`, `darwin/arm64`, `darwin/amd64`, `windows/amd64`.
- Preflight fails **before** any side effect: `make` tags only after all checks pass; the manifest is written once, last, after uploads succeed.
- Test command: `PERL5LIB=local/lib/perl5 prove -l t/<file>` (single) / `prove -lr t` (full). Run from the repo root.
- Commit style: subject + body, no trailer block; auto-push after each commit.

```perl
# lib/Deploy/GoBin.pm
package Deploy::GoBin;

use Mojo::Base -base, -signatures;

has [qw(repo gobin secrets runner s3 log exec)];

sub _parse_semver ($v) {
    die "not valid semver: '$v'\n" unless defined $v && $v =~ /^(\d+)\.(\d+)\.(\d+)$/;
    return ($1, $2, $3);
}

sub semver_cmp ($a, $b) {
    my @a = _parse_semver($a);
    my @b = _parse_semver($b);
    return $a[0] <=> $b[0] || $a[1] <=> $b[1] || $a[2] <=> $b[2];
}

sub bump_semver ($current, $level) {
    my ($maj, $min, $pat) = _parse_semver($current);
    return "$maj.$min." . ($pat + 1) if $level eq 'patch';
    return "$maj." . ($min + 1) . ".0" if $level eq 'minor';
    return ($maj + 1) . ".0.0"          if $level eq 'major';
    die "unknown bump level: '$level' (want patch|minor|major)\n";
}

sub resolve_version ($self, %opt) {
    my $latest = $opt{latest};
    my $version = defined $opt{version}
        ? $opt{version}
        : bump_semver($latest // '0.0.0', $opt{bump} // 'patch');
    _parse_semver($version);   # validates or dies
    if (defined $latest && semver_cmp($version, $latest) <= 0) {
        die "version $version must be newer than the current latest $latest\n";
    }
    return $version;
}

1;
```

```perl
our @DEFAULT_TARGETS = qw(linux/amd64 linux/arm64 darwin/arm64 darwin/amd64 windows/amd64);

sub _targets ($self) {
    my $t = $self->gobin->{targets};
    return (ref $t eq 'ARRAY' && @$t) ? @$t : @DEFAULT_TARGETS;
}

sub goreleaser_yaml ($self, $version) {
    my $b = $self->gobin;
    my (%os, %arch);
    for my $t ($self->_targets) {
        my ($o, $a) = split m{/}, $t, 2;
        $os{$o} = 1; $arch{$a} = 1;
    }
    my $doc = {
        version      => 2,
        project_name => $b->{name},
        builds => [{
            id       => $b->{name},
            binary   => $b->{name},
            main     => $b->{main} // '.',
            env      => ['CGO_ENABLED=0'],
            ldflags  => ['-s -w -X ' . $b->{version_var} . '=' . $version],
            goos     => [sort keys %os],
            goarch   => [sort keys %arch],
        }],
        archives => [{ id => $b->{name}, formats => ['tar.gz'] }],
        checksum => { name_template => 'SHA256SUMS', algorithm => 'sha256' },
        signs    => [{
            id        => 'ed25519',
            artifacts => 'all',
            cmd       => 'gobin-sign',
            args      => ['${artifact}'],
        }],
    };
    return Dump($doc);
}

sub resolve_config_path ($self, $version, %opt) {
    my $dir = $opt{dir} // $self->repo;
    my $checked_in = path($dir, '.goreleaser.yaml');
    return ($checked_in->stringify, 0) if $checked_in->exists;
    my $gen = path($dir, '.goreleaser.generated.yaml');
    $gen->spew_utf8($self->goreleaser_yaml($version));
    return ($gen->stringify, 1);
}
```

Note: the `signs.cmd` (`gobin-sign`) is the ed25519 signer shim wired in Task 8; the generated config names it so GoReleaser calls it per artifact. `${artifact}` is GoReleaser's own template variable, intentionally left literal in the YAML.

### Task 4: `version.json` mutation + rollback re-point (pure)

**Files:**
- Modify: `lib/Deploy/GoBin.pm`

**Interfaces:**
- Produces (all operate on / return a plain manifest hashref, never touching the network):
  - `Deploy::GoBin::empty_manifest($name) -> \%m` - `{ name => $name, latest => undef, min_supported => undef, builds => {} }`.
  - `$gobin->manifest_add_build($m, version => $v, arches => \%per_arch, min_supported => $ms) -> $m` - inserts `builds{$v}` = the per-arch map (`{ "linux/amd64" => {url,sha256,sig}, ... }`), sets `latest => $v` and `min_supported => $ms`.

  - `$gobin->manifest_rollback($m) -> ($m, $previous_version)` - re-points `latest` to the next-highest-semver build key below the current latest; dies "no prior build" when there is none.

```perl
sub manifest_add_build ($self, $m, %a) {
    $m->{builds}{ $a{version} } = $a{arches};
    $m->{latest}        = $a{version};
    $m->{min_supported} = $a{min_supported};
    return $m;
}

sub _versions_desc ($m) {
    return sort { semver_cmp($b, $a) } keys %{ $m->{builds} };
}

sub manifest_prune ($self, $m, $retain) {
    my @keep = (_versions_desc($m))[0 .. $retain - 1];
    my %keep = map { $_ => 1 } grep { defined } @keep;
    $keep{ $m->{latest} } = 1 if defined $m->{latest};   # never drop latest
    delete $m->{builds}{$_} for grep { !$keep{$_} } keys %{ $m->{builds} };
    return $m;
}

sub manifest_rollback ($self, $m) {
    my @desc = _versions_desc($m);
    my ($prev) = grep { semver_cmp($_, $m->{latest}) < 0 } @desc;
    die "no prior build to roll back to\n" unless defined $prev;
    $m->{latest} = $prev;
    return ($m, $prev);
}
```

```perl
sub preflight_make ($self, %opt) {
    my @problems;
    my $git   = $opt{git}   // $self->_default_git;
    my $which = $opt{which} // \&_which;

    my $st = $git->('git status --porcelain');
    push @problems, 'working tree is not clean (commit or stash first)'
        if !$st->{ok} || ($st->{output} // '') =~ /\S/;

    my $version = eval { $self->resolve_version(%opt) };
    push @problems, ($@ =~ s/\n$//r) if $@;

    push @problems, "'go' not found on PATH"         unless $which->('go');
    push @problems, "'goreleaser' not found on PATH"  unless $which->('goreleaser');

    my $key = $self->gobin->{sign_key};
    push @problems, "signing key '$key' missing from conf/secrets.conf"
        unless $key && ($self->secrets // {})->{$key};

    return \@problems;
}

sub make ($self, %opt) {
    my $git = $opt{git} // $self->_default_git;
    if (my @p = @{ $self->preflight_make(%opt) }) {
        die "cannot make:\n" . join('', map { "  - $_\n" } @p);
    }
    my $version       = $self->resolve_version(%opt);
    my $min_supported = $opt{min_supported} // $opt{latest} // $version;

    my $tag = "v$version";
    my $t = $git->("git tag -a $tag -m 'Release $tag'");
    die "git tag failed: $t->{output}\n" unless $t->{ok};
    my $p = $git->("git push origin $tag");
    die "git push failed: $p->{output}\n" unless $p->{ok};

    my ($config, $generated) = $self->resolve_config_path($version, dir => $self->repo);
    my $env = { CGO_ENABLED => 0, GOBIN_SIGNING_KEY => $self->secrets->{ $self->gobin->{sign_key} } };
    my $run = $self->runner->run(dir => $self->repo, config => $config, env => $env);
    die "goreleaser failed: $run->{output}\n" unless $run->{ok};

    my $checksums = $self->_read_checksums(path($self->repo, 'dist'));
    my $meta = { version => $version, min_supported => $min_supported, checksums => $checksums };
    my $meta_path = path($self->repo, 'dist', 'gobin-meta.json');
    $meta_path->spew_utf8(encode_json($meta));

    return { version => $version, config => $config, generated => $generated,
             meta_path => "$meta_path" };
}

sub _read_checksums ($self, $dist) {
    my $sums = path($dist, 'SHA256SUMS');
    return {} unless $sums->exists;
    my %out;
    for my $line (split /\n/, $sums->slurp_utf8) {
        next unless $line =~ /^(\S+)\s+(\S+)$/;
        my ($sha, $file) = ($1, $2);
        next unless $file =~ /_([a-z0-9]+)_([a-z0-9]+)\.(?:tar\.gz|zip)$/;
        $out{"$1/$2"} = $sha;
    }
    return \%out;
}
```

```perl
    my $name = $self->_infer_service
        or die "Run inside a Go repo whose 321.yml has a gobin: block.\n";
    my $svc  = $self->config->service($name);
```

```perl
        push @arts, { file => "$f", name => $f->basename, os => $os, arch => $arch };
    }
    return @arts;
}

sub preflight_release ($self, $version) {
    my @problems;
    push @problems, "nothing built for $version - run '321 gobin make'"
        unless $self->_dist_artifacts($version);
    my $s = $self->secrets // {};
    push @problems, 'S3 credentials missing from conf/secrets.conf'
        unless $s->{s3_access_key_id} && $s->{s3_secret_access_key};
    return \@problems;
}

sub release ($self, %opt) {
    my $meta_path = path($self->repo, 'dist', 'gobin-meta.json');
    my $meta = $meta_path->exists
        ? Mojo::JSON::decode_json($meta_path->slurp_utf8) : {};
    my $version = $opt{version} // $meta->{version};
    die "no version: pass one or run '321 gobin make' first\n" unless $version;

    if (my @p = @{ $self->preflight_release($version) }) {
        die "cannot release:\n" . join('', map { "  - $_\n" } @p);
    }

    my $prefix   = $self->gobin->{s3}{prefix};
    my $checks   = $meta->{checksums} // {};
    my (%per_arch, @uploaded);
    for my $a ($self->_dist_artifacts($version)) {
        my $base = "$prefix/$version/$a->{name}";
        $self->s3->put(key => $base,          file => $a->{file},          content_type => 'application/gzip');
        $self->s3->put(key => "$base.sig",    file => "$a->{file}.sig",    content_type => 'application/octet-stream');
        push @uploaded, $base, "$base.sig";
        $per_arch{"$a->{os}/$a->{arch}"} = {
            url    => $base,
            sha256 => $checks->{"$a->{os}/$a->{arch}"},
            sig    => "$base.sig",
        };
    }

    my $manifest = $self->live_manifest // empty_manifest($self->gobin->{name});
    $self->manifest_add_build($manifest,
        version => $version, arches => \%per_arch,
        min_supported => $meta->{min_supported});
    $self->manifest_prune($manifest, $self->gobin->{retain} // 5);
    $self->s3->put(key => $self->_manifest_key, content => encode_json($manifest));

    # verify: re-fetch and HEAD every url + sig
    my $verified = 1;
    my $live = $self->live_manifest;
    for my $arch (keys %{ $live->{builds}{$version} }) {
        my $b = $live->{builds}{$version}{$arch};
        $verified = 0 unless $self->s3->head(key => $b->{url})->{ok};
        $verified = 0 unless $self->s3->head(key => $b->{sig})->{ok};
    }

    return { version => $version, uploaded => \@uploaded,
             live => $per_arch, verified => $verified };
}
```

```perl
# lib/Deploy/GoBin/S3.pm
package Deploy::GoBin::S3;

use Mojo::Base -base, -signatures;
use Path::Tiny qw(path tempfile);

has [qw(bucket creds)];
has exec => sub ($self) { \&_shell_exec };

sub _env ($self) {
    my $c = $self->creds // {};
    return {
        AWS_ACCESS_KEY_ID     => $c->{s3_access_key_id}     // '',
        AWS_SECRET_ACCESS_KEY => $c->{s3_secret_access_key} // '',
    };
}

sub put ($self, %a) {
    my $bucket = $self->bucket;
    my $cmd = "aws s3api put-object --bucket $bucket --key $a{key}";
    if (defined $a{file}) { $cmd .= " --body $a{file}" }
    else {
        my $tmp = tempfile();
        $tmp->spew_raw($a{content} // '');
        $cmd .= " --body $tmp";
        $a{_tmp} = $tmp;   # keep alive until exec returns
    }
    $cmd .= " --content-type $a{content_type}" if $a{content_type};
    my $r = $self->exec->(cmd => $cmd, env => $self->_env, dir => undef);
    return { ok => $r->{ok} ? 1 : 0 };
}

sub head ($self, %a) {
    my $r = $self->exec->(
        cmd => "aws s3api head-object --bucket @{[$self->bucket]} --key $a{key}",
        env => $self->_env, dir => undef);
    return { ok => $r->{ok} ? 1 : 0 };
}

sub get ($self, %a) {
    my $out = tempfile();
    my $r = $self->exec->(
        cmd => "aws s3api get-object --bucket @{[$self->bucket]} --key $a{key} $out",
        env => $self->_env, dir => undef);
    return { ok => 0, content => undef } unless $r->{ok};
    return { ok => 1, content => $out->slurp_raw };
}

sub _shell_exec (%a) {
    local %ENV = (%ENV, %{ $a{env} // {} });
    my $cmd = $a{cmd};
    $cmd = "cd $a{dir} && $cmd" if $a{dir};
    my $output = `$cmd 2>&1`;
    return { ok => ($? == 0 ? 1 : 0), output => $output };
}

1;
```
